# Technical Documentation — Agent Guide

Implementation details, formulas, and reference material for developers.

## Documents

| Document | Scope |
|----------|-------|
| `ARCHITECTURE.md` | Detailed system inventory — every module, its responsibility, dependencies, file paths |
| `CORE_FORMULAS.md` | Mathematical formulas used in game systems — damage, extraction rates, belt throughput, power calculations |
| `REFERENCE_BUILDS.md` | Example bot/building configurations for testing and balance verification |

## When to Use

- **"How is module X wired to module Y?"** → `ARCHITECTURE.md`
- **"What formula calculates damage?"** → `CORE_FORMULAS.md`
- **"What's a balanced bot loadout for testing?"** → `REFERENCE_BUILDS.md`
