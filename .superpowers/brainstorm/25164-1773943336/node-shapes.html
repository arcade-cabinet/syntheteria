Radial Node Shapes — Information Density

Same radial layout, but nodes that can actually hold content. All include hover tooltips.

A. Rounded Rectangles

Icon + label (row 1) + stat/cost summary (row 2). Hover shows tooltip with full description. Most readable. Center stays circular for identity.

B. Pill Capsules

Compact horizontal pills. Icon circle on left, label + one stat line on right. Softer, more sci-fi feel. Less room than rectangles but tighter layout.

C. Wide Hexagons

Thematic hex badges — wide enough for icon + label on row 1, stat on row 2. Strategy-game feel. Center stays circular. Good balance of shape language and info density.

Hover Tooltip (all options)

Regardless of shape, hovering any node shows an attached tooltip with:

  • Actions: Full description, AP/MP cost, cooldown, prerequisites
  • Tech nodes: Research cost, turns remaining, prerequisite chain, unlock description
  • Robot classes: Base stats, material costs, build time, available tracks

Tooltip attaches below/beside the hovered node, stays within viewport bounds.
